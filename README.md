> A custom document and image management module built on Frappe Framework.

---

## Overview

This project implements a structured **image and document uploader** designed for workflow-driven applications. It supports organized uploads, previews, downloads, and permission-aware access.

---

## Key Features

- Folder-based image organization
- Bulk upload and download
- Role-based visibility
- Workflow-triggered status updates
- Optimized for ERPNext integrations

---

## Tech Stack

- Python
- Frappe Framework
- ERPNext UI customizations
